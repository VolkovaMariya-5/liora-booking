# План разработки: Liora

> Мульти-тенантная SaaS-платформа онлайн-записи.
> Стек: Next.js + NestJS 10 + PostgreSQL + Prisma + NextAuth.js v5 + Tailwind + shadcn/ui

---

## Структура проекта

```
Итоговый проект/
  /backend     ← NestJS 10
  /frontend    ← Next.js (App Router)
  ТЗ_Liora.md
  ПЛАН_РАЗРАБОТКИ.md
```

### 3. Backend: создать `ManageModule` ← 30 мин
Ключевой модуль. Frontend вызывает:
- `GET /manage/staff`, `POST /manage/staff`, `PATCH /manage/staff/:id`
- `GET /manage/services`, `POST /manage/services`, `PATCH /manage/services/:id`, `DELETE /manage/services/:id`
- `GET /manage/settings`, `PATCH /manage/settings`

ManageController делегирует вызовы в существующие `StaffService`, `ServicesService`, `BusinessesService`.
Все роуты защищены `@Roles(Role.BUSINESS_ADMIN)`.

### 4. Backend: создать `UploadModule` ← 20 мин
