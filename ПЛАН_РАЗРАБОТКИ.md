# План разработки: Liora

> Мульти-тенантная SaaS-платформа онлайн-записи.
> Стек: Next.js 14 + NestJS 10 + PostgreSQL + Prisma + NextAuth.js v5 + Tailwind + shadcn/ui

---

## Структура проекта

```
Итоговый проект/
  /backend     ← NestJS 10
  /frontend    ← Next.js 14
  ТЗ_Liora.md
  ПЛАН_РАЗРАБОТКИ.md
```

**Цель:** все API эндпоинты работают и задокументированы в Swagger.

- [ ] **ReviewsModule** — создание после COMPLETED + публичный список
- [ ] **FavoritesModule** — добавить/убрать/список
- [ ] **ProfileModule** — просмотр/редактирование/смена пароля/удаление
- [ ] **UploadModule** — загрузка в Cloudinary
- [ ] **AdminModule** — статистика и управление (SUPER_ADMIN)

> Для каждого модуля: Swagger декораторы + DTO с валидацией + RolesGuard + изоляция тенантов
